# RandomizedAlphaTest
This repository contains the replication files for the paper "A general randomized test for Alpha", written by Daniele Massacci, Lucio Sarno, Lorenzo Trapani, and Pierluigi Vallarino. Please contact Pierluigi Vallarino for any enquiry.

The folder "Monte Carlo" contains contains codes and simulated data to replicate Tables 4.1 to 4.6 and Figure 4.1.
The folder "Empirics" contains codes and data to replicate Table 5.1 and Figure 5.1.
