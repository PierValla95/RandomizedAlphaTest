# RandomizedAlphaTest
This repository contains the replication files for the paper "A general randomized test for Alpha", written by Daniele Massacci, Lucio Sarno, Lorenzo Trapani, and Pierluigi Vallarino. Please contact Pierluigi Vallarino for any enquiry.

The folder "Monte Carlo, main body" contains  codes and simulated data to replicate Tables 4.1 to 4.6 and Figure 4.1.
The folder "Empirics" contains codes and data to replicate Table 5.1 and Figure 5.1.

All folders have their own README file that explains their functioning.

The replication package relies on the R package RandAlphaTest. The package can be installed by downloading the tarball file contained in the R_package folder. Installation then runs with usual install.packages() function. This version of the package was last updated on April 16, 2026, using  version 4.4.3 of R.
