# RandomizedAlphaTest
This repository contains the replication files for the paper "A general randomized test for alpha".

The folder "Monte Carlo, main body" contains  codes and simulated data to replicate Tables 4.1 to 4.6 and Figure 4.1.
The folder "Empirics" contains codes and data to replicate Table 5.1 and Figure 5.1.

All folders have their own README file that explains their functioning.

The replication package relies on the R package RandAlphaTest. The package can be installed by downloading the source file (extension, .tar.gz) contained in the R_package folder. Installation then runs with the usual install.packages() function, with the appropriate command for installing from a source file. This version of the package was last updated on April 16, 2026, using  version 4.4.3 of R. Its installation requires the packages "Rtools" and "Rcpp".
